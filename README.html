ircredirect

Tool created by Ahecht. Source available at https://ircredirect.toolforge.org/source.php

The ircredirect tool redirects users to an irc channel on kiwiirc.com with a randomly generated username. The generated username consists of an adjective taken from a list of 20 adjectives, a noun taken from a list of nouns, and a two-digit number. The two digit number and the adjective are issued sequentially, while the noun is chosen randomly from the list.

Per https://phabricator.wikimedia.org/T172612, user consent must be obtained before redirecting to a third-party server. If user consent was gained on the page linking to the ircredirect tool, add consent=yes to the end of the URL to silenty redirect to kiwiirc.com without displaying the disclaimer, e.g. https://ircredirect.toolforge.org/?consent=yes

By default, the script redirects to the Wikipedia Help Channel (wikipedia-en-help on irc.libera.chat). This can be overridden by adding the channel= and server= parameters to the url, e.g. https://ircredirect.toolforge.org/?server=irc.libera.chat&channel=wikipedia-en-help

By default, the seed values are taken from at Wikipedia:IRC help disclaimer/nicks.json. This can be changed by adding a baseurl= parameter to the url. The URL should be urlencoded and must begin with "https", e.g. https://ircredirect.toolforge.org/?baseurl=https%3A%2F%2Fen.wikipedia.org%2Fw%2Findex.php%3Ftitle%3DWikipedia%3AIRC_help_disclaimer%2Fnicks%3Faction%3Draw

Seed value files will only be loaded from the following WMF-owned domains:

  • mediawiki.org
  • toolforge.org
  • wikibooks.org
  • wikidata.org
  • wikimedia.org
  • wikinews.org
  • wikipedia.org
  • wikiquote.org
  • wikisource.org
  • wikiversity.org
  • wikivoyage.org
  • wiktionary.org

The seed value file is a JSON file, and should contain a "nouns" object and an "adjectives" array. If the file is not found or not properly formatted, the generated name will default to "WPHelp" followed by a two digit number. An example seed file is shown below:

{ "_readme": "This page is used to generate the usernames created by the ircredirect tool. Please see the documentation at https://ircredirect.toolforge.org/README.html before editing.", "_nouns": "The below 'nouns' object contains animal or fruit names, one for each letter of the alphabet.", "nouns": { "A": "Alpaca", "B": "Berry", "C": "Canary", "D": "Durian", "E": "Emu", "F": "Fig", "G": "Grape", "H": "Heron", "I": "Ibex", "J": "Jaguar", "K": "Kiwi", "L": "Lemur", "M": "Melon", "N": "Narwhal", "O": "Ocelot", "P": "Papaya", "Q": "Quetzal", "R": "Rhino", "S": "Stork", "T": "Tiger", "U": "Unicorn", "V": "Vervet", "W": "Walnut", "X": "Xerus", "Y": "Yew", "Z": "Zebra" }, "_adjectives": "The below 'adjectives' array contains color and gemstone names, each with a unique starting letter. '''This list must contain exactly 20 items.'''", "adjectives": [ "Amber", "Brown", "Cyan", "Diamond", "Emerald", "Green", "Indigo", "Jade", "Khaki", "Lime", "Magenta", "Navy", "Orange", "Purple", "Red", "Silver", "Tan", "White", "Yellow", "Zircon" ] }

The script will return the text of the redirect URL and other debugging information instead of actually performing the redirect if the debug= parameter is set to any value.
